# instalacion de nodeJs
	- ## windows Y mac
		- [Descarga de nodeJS](https://nodejs.org/en/download)
	- ## linux
	- ```apl
	  debian distros
	  
	  curl -fsSL https://deb.nodesource.com/setup_20.x | bash - &&\
	  apt-get install -y nodejs
	  
	  ```
	- ```apl
	  arch distro
	  
	  pacman -S nodejs npm
	  ```
